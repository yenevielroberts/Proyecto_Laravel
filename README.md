# Proyecto con Laravel-Tienda online de comida Kntina

He desarrollado una apliación web como prueba tecnica para la empresa de Abalit.
La aplicación es sobre una tienda online de comida. 

## Funcionalidades:

1. Ver los productos
2. Agregar los productos a una cesta
3. Eliminar los productos de la cesta
4. Agregar diferentes cantidades de un mismo producto
5. Iniciar y cerrar sesión
6. Página de busqueda
